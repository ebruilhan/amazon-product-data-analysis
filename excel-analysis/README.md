# Excel Exploratory Data Analysis

This folder contains the data analysis performed using **Microsoft Excel** on the Amazon product dataset.

The goal of this analysis is to explore product category trends, customer ratings, review volumes and discount strategies using Excel-based analytical techniques.

The dataset was analyzed using **Pivot Tables, aggregation functions and data visualizations** to extract meaningful insights from the data.

---

## Dataset Overview

The dataset contains Amazon product information including:

- Product category hierarchy
- Product ratings
- Number of reviews
- Discount percentage
- Pricing information

These variables allow us to understand customer satisfaction levels and product popularity across different product categories.

---

# Analysis Techniques Used

The following Excel techniques were used during the analysis:

- Pivot Tables
- Aggregations (Average, Sum)
- Category-based grouping
- Data visualization
- Chart-based analysis

Excel was used to quickly explore patterns in the dataset and identify trends in ratings, reviews and discounts.

# Insights

From this analysis we can observe several patterns:

- Electronics accessories such as **USB cables and headphones** generate very high review volumes.
- Some kitchen appliance categories tend to receive **lower ratings**, which may indicate customer dissatisfaction.
- Certain product categories apply **very high discount percentages**, suggesting intense competition among sellers.
- High review counts often correlate with **high-demand consumer electronics products**.
