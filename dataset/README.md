# Dataset

This folder contains the dataset used for the analysis.

Dataset: **Amazon Product Dataset**

The dataset includes information about various Amazon products and their customer interactions.  
It is used to analyze product performance, customer satisfaction and pricing strategies.

---

## Dataset Features

The dataset contains the following variables:

- **product_category** — category of the product
- **rating** — average customer rating
- **rating_count** — number of customer reviews
- **price** — product price
- **discount_percentage** — discount applied to the product

These features allow us to analyze **customer engagement, product popularity and pricing behavior**.

---

## Purpose of the Dataset

This dataset is used for **exploratory data analysis (EDA)** using multiple tools:

- **Excel** for initial exploration and pivot table analysis
- **SQL** for querying and aggregating product data
- **Python (Pandas & Matplotlib)** for data analysis and visualization

The goal is to identify patterns in:

- customer ratings
- product demand
- review distribution
- discount strategies
